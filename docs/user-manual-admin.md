# User Manual - Admin Role

## Overview
Sebagai Admin di Redline Academy LMS, Anda memiliki akses penuh ke semua fitur sistem. Anda dapat mengelola pengguna, kursus, pendaftaran, pembayaran, pengumuman, dan melihat laporan analitik. Dashboard ini dibagi menjadi bagian yang dapat diakses oleh Trainer dan Admin, serta bagian khusus Admin.

## Akses Dashboard
1. Login ke sistem dengan akun Admin.
2. Akses dashboard melalui URL: `https://redlineacademy.com.au/pages/dashboard-admin.html`
3. Sidebar navigasi akan menampilkan menu-menu yang tersedia.

## Menu Navigasi

### Bagian Bersama (Trainer & Admin)
1. **Dashboard**
   - **Deskripsi**: Halaman utama yang menampilkan ringkasan aktivitas, statistik siswa, kursus, dan notifikasi.
   - **Fitur**:
     - Statistik: Jumlah siswa aktif, kursus, penilaian tertunda, dll.
     - Aktivitas Terbaru: Daftar aktivitas siswa dan kursus.
     - Pengumuman: Pesan penting dari admin.
   - **Cara Menggunakan**:
     - Klik ikon rumah di sidebar untuk kembali ke dashboard.
     - Lihat statistik untuk memantau performa.

2. **My Students**
   - **Deskripsi**: Kelola siswa yang ditugaskan kepada Anda (jika Trainer) atau semua siswa (jika Admin).
   - **Fitur**:
     - Daftar siswa dengan status (aktif, tidak aktif, berisiko).
     - Filter berdasarkan kursus atau status.
     - Lihat detail siswa: profil, kemajuan kursus, nilai.
   - **Cara Menggunakan**:
     - Klik "My Students" di sidebar.
     - Gunakan filter untuk mencari siswa.
     - Klik nama siswa untuk melihat detail dan mengedit.

3. **Course Management**
   - **Deskripsi**: Kelola kursus yang Anda ajar atau semua kursus (Admin).
   - **Fitur**:
     - Buat kursus baru.
     - Edit kursus: modul, materi, tugas.
     - Assign trainer ke kursus.
     - Lihat daftar kursus dengan statistik pendaftaran.
   - **Cara Menggunakan**:
     - Klik "Course Management".
     - Klik "Tambah Kursus" untuk membuat baru.
     - Edit kursus dengan mengklik judul kursus.

4. **Assignments & Grading**
   - **Deskripsi**: Kelola tugas siswa dan beri nilai.
   - **Fitur**:
     - Daftar tugas tertunda penilaian.
     - Lihat submission siswa.
     - Beri nilai dan komentar.
     - Notifikasi untuk tugas baru.
   - **Cara Menggunakan**:
     - Klik "Assignments & Grading".
     - Pilih tugas dari daftar.
     - Lihat submission dan beri nilai.

5. **Schedule**
   - **Deskripsi**: Kelola jadwal kelas dan pertemuan.
   - **Fitur**:
     - Kalender jadwal.
     - Buat jadwal baru.
     - Edit jadwal: waktu, lokasi, peserta.
   - **Cara Menggunakan**:
     - Klik "Schedule".
     - Gunakan kalender untuk melihat jadwal.
     - Klik tanggal untuk menambah atau edit jadwal.

6. **Messages**
   - **Deskripsi**: Komunikasi dengan siswa, trainer, atau admin lain.
   - **Fitur**:
     - Inbox pesan.
     - Kirim pesan baru.
     - Riwayat percakapan.
   - **Cara Menggunakan**:
     - Klik "Messages".
     - Pilih penerima dan ketik pesan.

7. **Reports & Analytics**
   - **Deskripsi**: Lihat laporan performa siswa, kursus, dan sistem.
   - **Fitur**:
     - Grafik kemajuan siswa.
     - Laporan penyelesaian kursus.
     - Analitik penggunaan.
   - **Cara Menggunakan**:
     - Klik "Reports & Analytics".
     - Pilih jenis laporan dari dropdown.

### Bagian Khusus Admin
1. **User Management**
   - **Deskripsi**: Kelola semua pengguna sistem: siswa, trainer, admin, marketer.
   - **Fitur**:
     - Daftar pengguna dengan role.
     - Tambah pengguna baru.
     - Edit profil, role, status.
     - Reset password.
   - **Cara Menggunakan**:
     - Klik "User Management".
     - Klik "Tambah Pengguna" untuk membuat akun baru.
     - Edit pengguna dengan mengklik nama.

2. **Enrollments & Payments**
   - **Deskripsi**: Kelola pendaftaran kursus dan pembayaran.
   - **Fitur**:
     - Daftar pendaftaran dengan status pembayaran.
     - Konfirmasi pembayaran.
     - Lihat riwayat pembayaran.
     - Generate invoice.
   - **Cara Menggunakan**:
     - Klik "Enrollments & Payments".
     - Filter berdasarkan status.
     - Klik pendaftaran untuk konfirmasi.

3. **Announcements**
   - **Deskripsi**: Buat dan kelola pengumuman untuk semua pengguna.
   - **Fitur**:
     - Buat pengumuman baru.
     - Edit pengumuman.
     - Kirim ke grup tertentu (siswa, trainer, dll.).
   - **Cara Menggunakan**:
     - Klik "Announcements".
     - Klik "Buat Pengumuman".
     - Isi judul, isi, dan pilih penerima.

4. **Settings**
   - **Deskripsi**: Konfigurasi sistem.
   - **Fitur**:
     - Pengaturan umum: nama situs, email, dll.
     - Pengaturan LMS: durasi sesi, batas upload.
     - Backup dan restore.
   - **Cara Menggunakan**:
     - Klik "Settings".
     - Edit pengaturan dan simpan.

## Tips Penggunaan
- Gunakan sidebar untuk navigasi cepat.
- Periksa notifikasi badge untuk tugas tertunda.
- Logout dengan mengklik tombol logout di header.

## Troubleshooting
- Jika halaman tidak loading, refresh browser.
- Untuk masalah akses, hubungi support admin utama.
- Pastikan koneksi internet stabil untuk fitur real-time.
