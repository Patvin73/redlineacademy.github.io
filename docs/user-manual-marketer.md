# User Manual - Marketer Role

## Overview
Sebagai Marketer di Redline Academy, Anda bertanggung jawab memasarkan kursus ke sekolah-sekolah SMK. Dashboard ini membantu Anda mengelola sekolah yang telah Anda kunjungi, mengajukan klaim komisi, melihat laporan, dan panduan lengkap.

## Akses Dashboard
1. Login dengan akun Marketer.
2. Akses melalui URL: `https://redlineacademy.com.au/pages/dashboard-marketer.html`
3. Navigasi tab di header atas.

## Menu Navigasi (Tabs)

1. **Ringkasan (Overview)**
   - **Deskripsi**: Dashboard utama dengan statistik komisi dan aktivitas.
   - **Fitur**:
     - Total Komisi, Pending Pencairan, Siswa Terdaftar, Sekolah Aktif.
     - Tabel Aktivitas Terbaru.
     - Struktur Komisi referensi.
   - **Cara Menggunakan**:
     - Lihat statistik untuk monitor performa.
     - Klik aktivitas untuk detail.

2. **Sekolah (Schools)**
   - **Deskripsi**: Kelola daftar sekolah yang telah Anda fasilitasi presentasi.
   - **Fitur**:
     - Tambah sekolah baru.
     - Edit detail sekolah (nama, alamat, tanggal presentasi).
     - Tabel daftar sekolah dengan status.
   - **Cara Menggunakan**:
     - Klik "Sekolah".
     - Klik "Tambah Sekolah" untuk input baru.
     - Edit dengan mengklik baris sekolah.

3. **Ajukan Klaim (Claim)**
   - **Deskripsi**: Form untuk mengajukan klaim komisi berdasarkan presentasi dan pendaftaran siswa.
   - **Fitur**:
     - Pilih sekolah dan tanggal presentasi.
     - Input jumlah siswa yang daftar.
     - Kalkulator komisi otomatis (Access Fee + Komisi Pendaftaran).
     - Submit klaim.
   - **Cara Menggunakan**:
     - Klik "Ajukan Klaim".
     - Isi form dengan data akurat.
     - Klik "Ajukan Klaim" setelah verifikasi.

4. **Laporan Komisi (Reports)**
   - **Deskripsi**: Riwayat semua klaim yang telah diajukan.
   - **Fitur**:
     - Tabel klaim dengan status (Pending, Approved, Rejected).
     - Klik untuk lihat detail klaim.
     - Breakdown komisi.
   - **Cara Menggunakan**:
     - Klik "Laporan Komisi".
     - Klik klaim untuk lihat detail dan status.

5. **Panduan (Guide)**
   - **Deskripsi**: Panduan langkah demi langkah untuk mengajukan klaim.
   - **Fitur**:
     - Step-by-step guide: Persiapkan presentasi, Lakukan presentasi, Ajukan klaim, dll.
     - Struktur komisi detail.
     - FAQ dan tips.
   - **Cara Menggunakan**:
     - Klik "Panduan".
     - Ikuti langkah-langkah untuk proses klaim yang benar.

## Fitur Tambahan
- **Language Switcher**: Ganti bahasa ID/EN.
- **Logout**: Keluar dari dashboard.

## Tips Penggunaan
- Pastikan data sekolah dan presentasi akurat sebelum ajukan klaim.
- Gunakan panduan untuk memahami struktur komisi.
- Periksa status klaim secara berkala.

## Troubleshooting
- Jika klaim ditolak, periksa feedback dari admin.
- Untuk bantuan, hubungi admin melalui pesan di dashboard.
