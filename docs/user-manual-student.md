# User Manual - Student Role

## Overview
Sebagai Student di Redline Academy LMS, Anda dapat mengakses kursus yang Anda ikuti, mengerjakan tugas, melihat jadwal, mendapatkan sertifikat, dan berkomunikasi dengan trainer. Dashboard ini dirancang untuk memudahkan pembelajaran Anda.

## Akses Dashboard
1. Login ke sistem dengan akun Student.
2. Akses dashboard melalui URL: `https://redlineacademy.com.au/pages/dashboard-student.html`
3. Sidebar navigasi akan menampilkan menu-menu yang tersedia.

## Menu Navigasi

1. **Dashboard**
   - **Deskripsi**: Halaman utama dengan ringkasan pembelajaran Anda.
   - **Fitur**:
     - Statistik: Kursus aktif, tugas tertunda, progress.
     - Aktivitas Terbaru: Update dari kursus.
     - Pengumuman: Pesan penting.
   - **Cara Menggunakan**:
     - Lihat overview progress Anda.
     - Klik link untuk akses cepat ke kursus atau tugas.

2. **My Courses**
   - **Deskripsi**: Daftar kursus yang Anda ikuti.
   - **Fitur**:
     - Lihat materi kursus.
     - Progress penyelesaian.
     - Akses video, dokumen, dll.
   - **Cara Menggunakan**:
     - Klik "My Courses".
     - Pilih kursus untuk mulai belajar.
     - Tandai modul sebagai selesai.

3. **Assignments**
   - **Deskripsi**: Kelola tugas dari kursus.
   - **Fitur**:
     - Daftar tugas aktif dan deadline.
     - Upload submission.
     - Lihat nilai dan feedback dari trainer.
   - **Cara Menggunakan**:
     - Klik "Assignments".
     - Klik tugas untuk melihat detail.
     - Upload file dan submit.

4. **Schedule**
   - **Deskripsi**: Lihat jadwal kelas dan deadline.
   - **Fitur**:
     - Kalender pertemuan.
     - Reminder tugas.
   - **Cara Menggunakan**:
     - Klik "Schedule".
     - Lihat tanggal penting.

5. **Certificates**
   - **Deskripsi**: Akses sertifikat yang telah Anda peroleh.
   - **Fitur**:
     - Daftar sertifikat.
     - Download PDF.
   - **Cara Menggunakan**:
     - Klik "Certificates".
     - Download sertifikat setelah menyelesaikan kursus.

6. **Messages**
   - **Deskripsi**: Komunikasi dengan trainer.
   - **Fitur**:
     - Inbox pesan.
     - Kirim pertanyaan ke trainer.
   - **Cara Menggunakan**:
     - Klik "Messages".
     - Pilih trainer dan kirim pesan.

7. **Profile**
   - **Deskripsi**: Kelola profil pribadi.
   - **Fitur**:
     - Edit informasi pribadi.
     - Ubah password.
   - **Cara Menggunakan**:
     - Klik "Profile".
     - Update data dan simpan.

## Tips Penggunaan
- Selesaikan tugas tepat waktu untuk menghindari penalti.
- Gunakan Messages untuk bertanya jika bingung.
- Periksa Schedule untuk mengatur waktu belajar.

## Troubleshooting
- Jika kursus tidak muncul, hubungi trainer atau admin.
- Untuk masalah login, reset password melalui link lupa password.
